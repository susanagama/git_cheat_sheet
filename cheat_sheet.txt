



Setup Git
$ git config --global user.name "<name>"
$ git config --global user.email <email>



Initialize git repo
$ git init



Add file to git stage
$ git add <nomeficheiro>



See what's on the git stage/Project status
$ git status



Commit the files that are on the git stage
$ git commit
	options
	 -m "message" // to insert the message
	 -a		// add files to stage


Ignore files that are on the stage / Create a file called .gitignore
$ vim .gitignore
and then add to the git stage



Check repo history
$ git log 
	options
	--graph 	// branch graph
	--oneline
	-p		// changes
	-1		// number of commits
	--decorate


Creat alias
$ git config --global alias.lg 'log --graph --oneline --decorate --all'




$ git commit -a



See the specifications of every file that is committed
$ git log



Differences
$git diff
	options
	--staged 	// changes between stage and last commit


Reset local changes
$git checkout
	options 
	<filename> 	// unstage file


Remove commit
$ git reset <commit>
	options 
	--hard 		// remove changes


Amend last commit
$ git commit --amend



Create branch
$ git branch <branchname>


Change to branch
$ git checkout <branchname>


Check current branch
$ git branch
